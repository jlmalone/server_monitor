GUI Installation

  1. Download the latest DMG from GitHub Releases
  2. Drag Server Monitor to Applications
  3. Launch and add your services

CLI Installation

# Clone the repository
git clone https://github.com/jlmalone/server_monitor.git
cd server-monitor/cli

# Install and link
npm install
npm link

# Start using
sm list
sm add --name "My App" --path ~/myapp --port 3000 --cmd "npm run dev"
sm start "My App"

üìã CLI Commands

sm list              # List all services with status
sm status [name]     # Detailed health check
sm start <name>      # Start a service
sm stop <name>       # Stop a service
sm restart --all     # Restart all services
sm logs <name>       # Tail service logs
sm add [options]     # Add new service
sm remove <name>     # Remove a service
